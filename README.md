# react-simple-table-project

**Installation**
-----------------
1. npm install --save-dev @types/react-dom

2. npm install simple-table-core

**Start server**
----------------
npm start

Optional: clear cache
---------------------
rm  node_modules/.cache

Reference - https://www.simple-table.com/
